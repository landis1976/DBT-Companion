# DBT Companion

A Dialectical Behavior Therapy daily companion app for mood tracking, 
diary cards, DBT skills reference, and crisis support.

## Features
- Daily diary cards
- Mood tracking
- DBT skills reference
- Crisis support tools

## Privacy
All data is stored locally in your browser. 
Nothing leaves your device.
